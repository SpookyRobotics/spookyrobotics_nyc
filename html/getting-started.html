An Open Source Hardware Company

Engineering Mission

SpookyRobotics is intended to be an open source business that educates makers, programmers, and engineers to build electromechanical devices. The products that this company builds will be manufactured using open source tools whenever possible. To accomplish this goal, we must establish a baseline toolset that engineers are expected to use. An engineer must seek mastery of the following

  • Software Engineering
    The ability to design a system that uses code, for example on a whiteboard

  • Programming
    The ability to take a given design and implement it in a programming language

  • Computer-aided Design
    The ability to model a physical product on a computer, for display to a non-technical individual. The model should be constructed in a way that it can be produced on a consumer grade manufacturing machine.

  • Consumer Level Manufacturing
    The ability to use a consumer manufacturing machine. This includes 3D printers, CNC Machines, Laser Cutters, and any other automated process that manipulates physical objects to modify their resulting shape.

  • Circuit Design
    The ability to read a schematic, solder and diagnose errors in an electrical system.
I believe the above skill set can be self-taught to a level acceptable enough for generating an income stream. Hence this company will create the necessary documentation for our products to be understood by any individual with the above skills. When possible, we will strive to create educational material to facilitate learning the necessary material to build our products.

Development Environment

Since there are many options for open source tools, we specify the software and hardware an engineer must be able to use. They are the following:

  • Inkscape
  • FreeCad
  • Ubuntu Linux
  • Raspberry Pi
  • Beagleboard
  • Arduino
  • Any programming language environment that runs on Ubuntu
